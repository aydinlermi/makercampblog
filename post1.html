Gün 1: Web sitesi oluşturma

Web sitesi hazırlayabilmek için gereken CSS ve HTML bilgilerini öğreniyoruz!

22 Ocak 2018 tarihinde Aydın Emre Lermi tarafından paylaşıldı.

İlk gün! Türkiyenin her yerinden öğrenciler, tecrübe ve bilgi kazanmak için Özyeğin Maker kampına geliyor!

İlk günümüzde Erdem hoca bize bir web sitesi kodlayabilmek için gerekli bilgileri öğretiyor. Ne gibi mi? Sonundan başına kadar herşeyi. Kullanılabilecek dilleri, etiketleri, terimleri... Html Web siteleri HTML denilen, kısmi bir dilden oluşuyormuş. Sadece HTML kullanarak basit siteler yazılabiliniyormuş, bunun yanında bir de CSS varmış. Css HTML'i Geliştirip, ona bir stil ve "hava" katmayı sağlıyormuş.

Basit bilgileri öğrendikten sonra biraz dalga geçmeye başladık, saçma şeyler hakkında web siteleri hazırlayıp çalıştırdık. Ardından, bir yemek sitesi hazırlamayı denedik, bir çok uğraşla, herkes çalışmayı zamanında bitirebildi!
